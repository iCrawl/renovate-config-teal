## Install

```bash
pnpm i -D renovate-config-teal
```

## Usage

Add the Renovate config to your `package.json`:

```json
{
	"name": "my-project",
	...
	"renovate": {
		"extends": ["teal"]
	}
}
```

Or to `renovate.json`:

```json
{
  "extends": ["teal"]
}
```
